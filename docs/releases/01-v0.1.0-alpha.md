# v0.1.0 Alpha Contract

This document freezes the current GoShip `v0.1.0-alpha` command and HTTP route surface.

## Frozen Surfaces

- root CLI help output from `ship`
- route inventory emitted by `ship routes --json`

These snapshots are enforced by `make test-alpha-contracts` in CI.

## Change Policy

Do not refresh the alpha snapshots as part of routine refactors.

A snapshot update is only allowed when:

1. the contract change is intentional and documented in the same PR
2. the PR includes approved review for the alpha surface change
3. the snapshot is refreshed explicitly with `UPDATE_ALPHA_CONTRACTS=1 make test-alpha-contracts`

That approved review must exist before merge, not after the snapshot update lands.

If the gate fails unexpectedly, treat it as contract drift until the review explicitly approves changing the frozen surface.
