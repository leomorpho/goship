# TKT-183: Delete unregistered/unreachable app routes, controllers, views, and assets

## Description
Perform destructive dead-code cleanup for unregistered or unreachable app routes/controllers/views/assets.

Context:
Repo contains stale artifacts and demo leftovers not part of active HTTP/runtime surface.

Scope:
- Build reachability list from router/module registration.
- Delete unreachable code/assets with proof of non-use.

Out of scope:
- Removing live feature paths.

Likely paths:
- `app/router.go`
- `modules/**/routes*.go`
- `app/web/controllers/**`
- `app/views/**`
- `app/static/**`

Verify commands:
- `go run ./tools/cli/ship/cmd/ship -- routes`
- `go test ./app/web/controllers/... ./app/...`
- `go test ./...`

## Acceptance Criteria
- [ ] Reachability audit identifies deletions with router/module registration proof.
- [ ] Unreachable artifacts are removed without breaking compile/runtime registration.
- [ ] No live route handlers are deleted.
- [ ] Deletion log in ticket notes lists each removed artifact with reachability proof.
- [ ] Commit message records deletion-safety evidence for dead-code sweep.
