# TKT-148: Dynamic `<html lang>` propagation across layouts/pages

## Description
Replace hardcoded `lang="en"` in rendered HTML surfaces with request-resolved locale.

## Hierarchy
Parent: `TKT-143`
Blocks: `TKT-146`
Related prior work: `TKT-121`

## Scope
1. Identify all layouts/pages/components with hardcoded `<html lang>` (and equivalent roots).
2. Thread request locale into templ roots consistently.
3. Add regression tests for at least `en` + `fr` rendering.

## Acceptance Criteria
- [ ] Core rendered HTML surfaces no longer hardcode `lang="en"`.
- [ ] Locale switching updates the rendered `lang` attribute deterministically.
- [ ] Tests cover multiple locales and fallback behavior.
